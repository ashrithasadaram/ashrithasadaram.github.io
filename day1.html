scope of project

The scope of the project includes the following key areas:

  • - Seat Allocation & Booking: Managing reservations, availability, and assignments.
  • -- System Integration: Connecting with databases, payment gateways, and scheduling tools.
  • --- User Interface Design: Creating an intuitive and user-friendly interface for both customers and administrators.
  • ---- Reporting & Analytics: Generating reports on bookings, user activity, and system performance.
  • ----- Data Visualization: Creating dashboards and visual representations of seats availability and usage.

also learnt about:

wireframe:

Wireframing is a practice used by UX designers which allows them to define and plan the information hierarchy of their design for a website, app, or product. This process focuses on how the designer or client wants the user to process information on a site, based on the user research already performed by the UX design team.

user research:

User research is a crucial part of the UX design process. It involves understanding user behaviors, needs, and motivations through observation and interviews. This information helps designers create more effective and user-centered designs.

git/github:

Git is a version control system that allows developers to track changes in their codebase. GitHub is a platform that hosts Git repositories, enabling collaboration among developers. It provides features like pull requests, issues, and project management tools to streamline the development process. and also helps to submit our daily reports.
